# Roger HR — AI Chief of Staff Application

## Tell us about a problem you solved that you are proud of. (1000 char max)

At Bazzile, the real estate SaaS I cofounded, I built an in-app voice agent that holds a conversation with the user and proposes the two or three properties that fit — not another paginated grid to scroll through.

The first version runs on Wonderful's agentic platform. I am rebuilding it on Elixir/Jido with Mistral's Voxtral for STT/TTS, to fully own the stack.

What looks simple took real engineering. A voice agent cannot consume the same APIs a UI does — paginated lists are useless when you need to narrate two or three options out loud. I designed agent-shaped APIs, tools tailored to what an LLM can speak about, real-time monitoring on every call, and a data layer that gives the agent context without drowning it.

It works as a real product feature, in friends-and-family testing. Building agents in production is not the LLM — it is the engineering around it.
